# 8.8 通知

## 入口与目标
通知不再作为左侧一级页面存在，统一收敛为顶栏右上角铃铛按钮 + 下拉通知面板。

> **说明：** 当前版本只保留轻量应用内通知入口；不做独立通知中心页，不做 drawer，不做长期归档体系。

## 面板结构
- 标题：通知
- 全部 / 未读 切换
- 全部已读
- 最近 20 条通知列表

## 当前版本仅保留的通知类型
- 审核进度相关
  - skill 已进入审核
  - skill 审核通过
  - skill 审核拒绝
  - skill 被退回修改
  - 你有新的待审核任务
- Skill 更新
  - 某个已安装 Skill 有新版本可更新
- 软件更新
  - 桌面客户端本身有新版本

## 不再展示的内容
- skill 权限已收缩
- 当前安装副本不可再更新
- 连接状态异常 / 恢复
- 工具或项目路径异常
- 安装 / 更新 / 卸载 / 启用 / 停用结果
- “同步本地事件”按钮
- 搜索、删除、独立“查看全部页面”

## 通知项字段
- 通知标题
- 通知摘要
- 图标
- 时间
- 已读 / 未读状态

## 点击行为
- 新的待审核任务 / 管理员审核任务 -> 跳转审核页
- 作者视角审核进度 -> 跳转“我发布的”页面
- Skill 更新 -> 跳转“我的 Skill”并优先切到“有更新”视图
- 软件更新 -> 打开轻量更新提示弹窗

## 面板操作
- 全部已读
- 点击单条通知后关闭面板
- 点击面板外区域关闭
- 再次点击铃铛关闭

## 功能范围

- 用户可手动切换
