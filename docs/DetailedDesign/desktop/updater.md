# updater 模块设计

## 模块目标
管理桌面客户端版本检查、升级包下载、升级提示与安装前校验。

## 主要职责
- 查询最新桌面版本
- 校验升级包签名/哈希
- 控制升级提示节奏
- 升级前检查 WebView2 与权限条件

## 本地状态
- `desktop_update_state`
- `desktop_update_history`

## 交互
- HTTP：获取最新版本与升级包元信息
- IPC：下载文件、触发安装程序

## 失败/恢复
- 下载失败支持断点续传或重试
- 升级包校验失败立即阻断安装

## 风险/权衡
- 内网环境下升级节奏要可控，支持管理员统一发版窗口
